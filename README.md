# BMW Sales Analysis using SQL

A SQL data analysis project focused on exploring **BMW Global Sales Data (2010–2024)**. The project answers real-world business questions using SQL and demonstrates practical skills in data exploration, aggregation, and business analytics.

##  Project Overview

This project analyzes BMW sales data to uncover sales trends, pricing patterns, regional performance, and vehicle profitability. Using SQL, the dataset was transformed into meaningful business insights that support data-driven decision-making.

##  Tech Stack

* **SQL**
* **PostgreSQL**
* **pgAdmin 4**

##  Business Problems Solved

| #  | Business Question                                                                          |
| -- | ------------------------------------------------------------------------------------------ |
| 1  | Find the **Top 10 BMW models** by total sales volume.                                      |
| 2  | Identify the **Top 5 regions** with the highest sales volume.                              |
| 3  | Calculate the **average price of Electric BMW vehicles**.                                  |
| 4  | Analyze **total sales volume and average price** for BMWs sold in **2017**.                |
| 5  | Determine the **highest-selling BMW model in each region**.                                |
| 6  | Create a summary of **High vs Low sales classifications** by region.                       |
| 7  | Compare the **average prices of Manual and Automatic transmission** vehicles.              |
| 8  | Analyze the relationship between **Engine Size and Average Sales Volume**.                 |
| 9  | Perform a **Depreciation Analysis** using average price and mileage by model year.         |
| 10 | Identify the **most profitable region for each BMW model** based on average selling price. |

##  Key Insights

* Identified the best-selling BMW models.
* Discovered the strongest sales regions.
* Evaluated Electric Vehicle pricing.
* Compared Manual and Automatic transmission prices.
* Analyzed sales performance for the year 2017.
* Found the top-performing model in every region.
* Compared High and Low sales classifications across regions.
* Explored how engine size impacts sales.
* Analyzed vehicle depreciation trends.
* Identified the most profitable market for each BMW model.
